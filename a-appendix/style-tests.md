# 格式化文本测试

## 使用 `<span>` 添加样式

```html
<span style="color:red;">红色文字</span>
```
<span style="color:red;">红色文字</span>

```html
<span style="border: medium solid red;">边框</span>
```
<span style="border: medium solid red;">边框</span>

## 使用 `<font>` 添加格式（不推荐）

```html
<font color="#f00;">红色文字</font>
```
<font color="#f00;">红色文字</font>

## 路径

```html
<span class="paths">%SystemRoot%\System32\\</span>
```

<span class="paths">%SystemRoot%\System32\\</span>

## 菜单

```html
<span class="menutext">Help</span>
```
<span class="menutext">Help</span>
